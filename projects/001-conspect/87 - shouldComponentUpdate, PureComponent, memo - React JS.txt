рендер несколько раз. Так происходит потому что не все props приходят сразу.

Для классовых компонент есть несколько способов чтобы избежать ререндера:

--- Первый способ это внутри компоненты перед render() использовать метод shoudComponentUpdate()

Выглядит так:
    shouldComponentUpdate(nextProps, nextState) {
        return nextProps != this.props || nextState != this.state;
    }

--- Второй способ исползовать вместо Component -> PureComponent

Выглядит это так:
const Profile = React.memo(props => {

    console.log(props);
