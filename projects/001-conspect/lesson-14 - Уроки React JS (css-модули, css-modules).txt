В уроке рассказано про инкапсуляцию стилей css
Допустим есть две разных компоненты которые имеют собственные файлы стилей и в обеих файлах
стилей есть .item. Будет несостыковочка. Поэтому нужна инкапсуляция.
1) Импортировать стили уже нужно так: import classes from './Knowledge.module.css';
2) Теперь для того чтобы присвоить класс пишем так: <li className={classes.item}>Знаю что такое "Компонента"</li>
3) Больще нельзя в названиях классов использовать девис, пример: "item-elem", так как потом прийдется 
писать так: className={classes.item-elem} а на это будет ругаться JS. (Неверно указанно свойство объекта)
4) Если нужно присобачить два класса пишем так: className={`${classes.item} ${classes.active}`}
